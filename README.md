# Click-Kill-Aliens
Juego en Pygame de Matar Aliens Con El Ratón

# Descripción
Al principio te pedirá que introduzcas un número del 0 al 9, ese será el número de enemigos 

# Controles Nave:
- *SPACE:* disparo
- *Flecha izquierda/derecha:* movimiento lateral
